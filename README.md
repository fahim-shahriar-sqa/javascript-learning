# JavaScript Learning for SQA Automation 🚀

This repository documents my structured learning journey in JavaScript with a strong focus on **Software Quality Assurance (SQA)**, including API testing, performance testing (K6), and automation frameworks.

The goal is to build a strong foundation in JavaScript for real-world test automation tools like:
- K6 Performance Testing
- Postman scripting
- API Automation frameworks (Node.js-based)
- Test utilities for backend validation

---

## 🎯 Learning Objectives

By completing this repository, I aim to:

- Understand core JavaScript concepts used in automation
- Write reusable functions for API testing
- Build and manipulate JSON payloads
- Handle API responses effectively
- Structure code like real automation frameworks
- Debug and handle errors in test scripts
- Prepare for real-world SQA engineering tasks
